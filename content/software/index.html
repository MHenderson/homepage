todotxt

An R package for parsing todo.txt files.

https://mhenderson.github.io/todotxt/

https://github.com/MHenderson/todotxt

funcgeo

An R package for Functional Geometry.

https://mhenderson.github.io/funcgeo/

https://github.com/mhenderson/funcgeo

vizing

A Python package that extends NetworkX to provide edge-, list- and total-colouring. It wraps the python-constraint package and provides conversions between graph-colouring and constraint satisfaction problems.

https://pythonhosted.org/vizing/

https://github.com/MHenderson/vizing

ryser

A Python package for latin square completion. It uses Vizing as a solving engine.

https://pythonhosted.org/ryser/

https://github.com/MHenderson/ryser

rminion

If you have a CSP in Minion format and only need to call Minion from R and get the result as text then you only need rminion.

https://minion-r.github.io/rminion/

https://github.com/minion-r/rminion

minionator

If, in addition, you also want to generate CSPs using the R programming language then you also need minionator.

https://minion-r.github.io/minionator/

https://github.com/minion-r/minionator

mopr

Finally, if you want to automatically parse the results of calling Minion into an R data frame then you also need mopr.

https://minion-r.github.io/mopr/

https://github.com/minion-r/mopr

sudoku

Python package for modelling Sudoku.

https://nbviewer.jupyter.org/gist/MHenderson/6416294

https://bitbucket.org/matthew/sudoku/src/default/

wave

Whole Architecture Verification.

https://pythonhosted.org/Wave/

https://gitlab.com/MHenderson1/wave

experts

Code to accompany Henderson, Shawe-Taylor, and Žerovnik (2005).

https://gitlab.com/MHenderson1/experts

psssodls.py

Generator of PSSSODLS in Minion format.

https://gist.github.com/MHenderson/375a2122f1b77f90e5df

psssodls.eprime

Essence’ description of PSSSODLS

https://gist.github.com/MHenderson/6200476
