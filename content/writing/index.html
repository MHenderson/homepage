Notes

  • On Polytopes and Lattice-Point Enumeration - Notes on Ehrhart Theory, including examples in Polymake.
  • On the Completability of Wing-Form Partial Latin Squares - Completable, bound, untethered KF-SPLS of even order are characterised.
  • Some Observations on the Intricacy of Partial Latin Square Completion - Some ideas about the intricacy of completing partial latin squares.
  • On the Completability of Kite-Form Partial Latin Squares - Characterising the completable bound, untethered KF-SPLS of even order.
  • On Pandiagonal Strongly-Symmetric Self-Orthogonal Diagonal Latin Squares - Describing the discovery of a PSSSODLS of order 9.
  • Constraint Modelling of Pandiagonal Strongly-Symmetric Self-Orthogonal Diagonal Latin Squares - Using Essence’, Savile Row and Minion to model PSSSODLS.
  • A Brief Introduction to Goldberg’s Conjecture - An introduction to Goldberg’s conjecture on edge-colouring.
  • Basic Text Analysis of “Jude the Obscure” by Thomas Hardy - Working through the book “Text Analysis with R for Students of Literature” by Matthew L. Jockers.
  • JAGS demo - Working through Chapter 8 of “Doing Bayesian Data Analysis” by John K. Kruschke.
  • Notes on Richard McElreath’s Statistical Rethinking - NA

Notebooks

  • Completing KF-SPLS with Ryser - Demonstration of completing KF-SPLS using the ryser Python package.
  • An Introduction to NetworkX - Demonstration of NetworkX for graph and network analysis in Python.
  • Computational and Numerical Methods - Using Octave for numerical methods.
  • Processing GPS Data in Python - A statistical investigation, based on GPS data, into anomalous driving patterns among a fleet of delivery trucks.
  • Modelling Sudoku Puzzles in Python - An updated version of Davis, Henderson, and Smith (2010).
  • Speech Annotation in Python Using Regular Expressions - Some basic ideas about annotating a corpus of novels using regular expressions.
  • List-colouring in Python with Vizing - Introducing list-colouring of graphs using the the vizing Python package.
